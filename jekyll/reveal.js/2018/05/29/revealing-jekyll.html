    Revealing Jekyll

    May 29, 2018

        As a Beamer alternative, I have been wanting to look into the Reveal.js presentation framework for sometime. Also, I thought it would be a nice way of spicing up some blog posts with a flashcard effect. To this end, we will cover how to configure and integrate the Reveal.js framework within your Jekyll site.

    Loading Reveal.js

        To integrate Reveal.js with Jekyll, we will need a copy of the framework within your site’s root directory.

    $ wget https://github.com/hakimel/reveal.js/archive/master.zip
$ unzip master.zip -d ./reveal.js
$ rm master.zip

    Or you can git clone the framework.

    $ git clone https://github.com/hakimel/reveal.js
$ rm -r reveal.js/.git
