### Reveal.js supports syntax highlighting with [Highlight.js](https://github.com/isagalaev/highlight.js/).

To enable, configure Reveal.js with the Highlight.js plugin.


      Reveal.initialize({
        dependencies: [
          { src: 'plugin/highlight/highlight.js', async: true,
              callback: function() { hljs.initHighlightingOnLoad(); }}
        ]
      });

Also, load a Highlight.js CSS theme.


      <link rel="stylesheet" href="/reveal.js/lib/css/zenburn.css"/>

Reveal.js uses MathJax to display mathematics.


      Reveal.initialize({
        math: {
          mathjax: 'https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.4/latest.js',
          config: 'TeX-AMS-MML_HTMLorMML'
        }

        dependencies: [
          { src: '/reveal.js/plugin/math/math.js', async: true }
        ]
      });
